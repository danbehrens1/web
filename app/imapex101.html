What is imapex 101?

Developing applications using modern technologies and practices like DevOps requires more than simply understanding a programming language like Python or Javascript. Experience and skill with source control, container technologies, collaboraiton services, and build bots is also needed.

imapex 101 is an intensive hands-on training program designed to jump start new developers with a foundation in DevOps practices. The training material is designed to introduce a concept, provide basic hands-on examples, and challenges to reinforce the material.

imapex 101 Modules

Module 1: General Development Skills Module 2: Advanced Git and GitHub Module 3: Advanced Docker and Docker Hub Module 4: Continuous Integration/Deployment Tools Module 5: Deploying to Mantl.io Module 6: Leveraging Cisco DevNet Module 7: Developing with Cisco Platforms Module 8: Taking Demo Idea to Application

Each module is designed to follow this basic flow:

  • Introduce a concept
  • Walk through some guided exercises
  • Provide links for more details
  • Answer the question "Why do we care?"
  • Include a series of challenges to explore and reinforce the topics

Finding the Content

All the content for imapex 101 is stored at github.com/imapex/imapex101.

The material has been, and continues to be, developed by the imapex community, and is stored in a public GitHub repository to make it easy for anyone to leverage, and improve. If you find a mistake, have a suggestion for improvement, or an idea for a new module, please fork our repository and send us a Pull Request.
